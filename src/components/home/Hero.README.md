# Hero

Kort beskrivelse: Helbredde toppseksjon med bakgrunnsbilde og hovedbudskap.

## Bruk
- Bruk på forsiden for å introdusere tjenesten.
- Bruk kort overskrift og støtte-tekst.

## Ikke bruk
- Ikke bruk på undersider med tett innhold.
- Ikke bruk lange tekster i hero.

## Retningslinjer for innhold
- Tittel på 4–8 ord.
- Én kort støttetekst på 1–2 setninger.

## Props (kort)
- Ingen props. Innholdet er fast definert.

## Eksempler
```tsx
<Hero />
```

## Tilgjengelighet
- Bakgrunnsbilde har `alt`-tekst.
- Sørg for tilstrekkelig kontrast mellom tekst og bakgrunn.

## Vedlikehold/Notater
- Endre tekster i komponenten ved behov for kampanjer.

## Gjør
- Hold budskapet kort og tydelig.
- Sørg for god kontrast.

## Unngå
- Ikke bruk lange avsnitt.
- Ikke legg inn flere hovedbudskap.

# SaunaStory

Kort beskrivelse: Historie- og inspirasjonsseksjon for badstukonseptet.

## Bruk
- Bruk på informasjonssider eller som innhold på badstuesider.

## Ikke bruk
- Ikke bruk på sider med strenge innholdsgrenser.

## Retningslinjer for innhold
- Teksten bør være rolig, fortellende og konsekvent i tone.
- Bruk korte avsnitt for lesbarhet.

## Props (kort)
- Ingen props. Innholdet er fast.

## Eksempler
```tsx
<SaunaStory />
```

## Tilgjengelighet
- Bruk semantiske overskrifter for struktur.

## Vedlikehold/Notater
- Oppdater tekstinnhold i komponenten ved behov.

## Gjør
- Hold teksten rolig og fortellende.
- Del opp i korte avsnitt.

## Unngå
- Ikke bruk veldig lange blokker.
- Ikke bland flere temaer uten struktur.

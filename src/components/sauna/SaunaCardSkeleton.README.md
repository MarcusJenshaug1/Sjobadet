# SaunaCardSkeleton

Kort beskrivelse: Lasteplassholder for badstuekortenes layout.

## Bruk
- Bruk mens badstuer lastes inn.

## Ikke bruk
- Ikke bruk når innholdet allerede er klart.

## Retningslinjer for innhold
- Skal etterligne kortets størrelse og struktur.

## Props (kort)
- Ingen props.

## Eksempler
```tsx
<SaunaCardSkeleton />
```

## Tilgjengelighet
- `aria-busy` og `aria-label` er satt.

## Vedlikehold/Notater
- Animasjon er definert inline.

## Gjør
- Bruk når data lastes.
- Hold samme størrelse som ekte kort.

## Unngå
- Ikke bruk når innholdet er klart.
- Ikke bruk for andre korttyper.

# Flater

Flater representerer bakgrunner og containere vi bruker for å skape hierarki.

## Sidebakgrunn
Brukt som hovedbakgrunn for alle sider.
<div style={{ padding: '2rem', backgroundColor: 'var(--background)', border: '1px solid var(--border)', borderRadius: 'var(--radius)' }}>
  Dette er standard sidebakgrunn (`--background`).
</div>

## Sekundær flate
Brukt for seksjoner som skal skille seg litt ut, som footer eller avgrensede bakgrunner.
<div style={{ padding: '2rem', backgroundColor: 'var(--secondary)', border: '1px solid var(--border)', borderRadius: 'var(--radius)', marginTop: '1rem' }}>
  Dette er en "Inset" flate (`--secondary`).
</div>

## Kort og elevasjon
Hvite containere på grå bakgrunn for å løfte innhold.
<div style={{ padding: '2rem', backgroundColor: 'white', borderRadius: 'var(--radius)', border: '1px solid var(--border)', boxShadow: '0 1px 3px 0 rgb(0 0 0 / 0.1)', marginTop: '1rem' }}>
  Dette er en standard kort-flate med subtil skygge.
</div>
